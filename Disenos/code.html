Guía de Exportación a Antigravity

Sigue estos pasos detallados para migrar tu diseño y lógica de negocio desde el entorno de CoreOps hacia la infraestructura de producción en Antigravity.

1. Exportación de Código

Obtén el código fuente completo basado en React 18 y TypeScript listo para desarrollo local o CI/CD.

  • A Selecciona ZIP Download para una copia local rápida de los archivos fuente.
  • B Usa Push to Git para vincular directamente con tu repositorio en GitHub o GitLab.

2. Sincronización de Base de Datos

Aplica el modelo de datos y las políticas de seguridad (RLS) en tu instancia de Supabase/Antigravity.

# Ejecuta el script de migración SQL
psql -h db.antigravity.io -f schema.sql
  • A Copia los scripts generados de RLS (Row Level Security).
  • B Ejecútalos en el Editor SQL de tu Dashboard de Antigravity.

3. Recursos y Estilos

Asegúrate de que la identidad visual se mantenga íntegra exportando las configuraciones de Tailwind.

tailwind.config.js
Assets Folder (/public)
Design Tokens (JSON)

4. Próximos Pasos

Configuración final antes de que tu aplicación esté en vivo.

  • Configura las Variables de Entorno (.env) con tus claves API de Antigravity.
  • Realiza un npm install y npm run build para verificar el bundle.
  • Despliega a producción usando Antigravity Cloud o tu provider preferido.
